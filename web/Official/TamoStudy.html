TamoStudy

TamoStudy is a free, focus/study application that implements a fun, virtual pet to help users stay driven to focus.
TamoStudy was created and developed by Anthony Narlock on January 30th, 2021.
Programmed in Java, a Java Runtime Environment must be installed to use.

  • Set a timer for how long you want to study.

  • Earn TamoTokens everytime you study.

  • Feed your Tamo and keep them happy!
